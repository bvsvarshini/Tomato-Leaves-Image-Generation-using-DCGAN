# 🍅 Tomato Leaf Disease Detection

## 🎯 Objective
To detect and classify different tomato leaf diseases using deep learning models.

## 📚 Project Overview
This project focuses on identifying multiple tomato leaf diseases using image-based classification. Each disease is handled using a separate deep learning notebook.

All notebooks were developed and executed using **Google Colab**, making it easy to run without local setup.

---

```## 🗂️ Project Structure

Tomato-Disease-Detection/
├── bacterialspot.ipynb
├── earlyblight.ipynb
```

## ▶️ How to Run
1. Open any `.ipynb` file in **Google Colab**  
2. Upload or mount dataset zip files  
3. Extract datasets  
4. Run all cells sequentially
